Informações sobre este problema

O navegador pode armazenar os dados em cache, fazendo com que alterações recentes não apareçam imediatamente. Este utilitário irá:

  • Limpar todos os caches do navegador relacionados a estes dados
  • Garantir que os dados de Jonathan Williams e outros pilotos estejam atualizados
